# Gllacy Shop 2018
Проект состоит из 2-х страниц index.htmt, catalog.html.

## Используется:
- Flex-box
- Gulp 
- Sass(scss)
- JavaScript
- Svg-images

## Дополнительно:
- При достижении 1200px сетка перестаёт масштабироваться дальше.
- Кроссбраузерность: IE11+, Chrome, Firefox, Opera, Safari.
